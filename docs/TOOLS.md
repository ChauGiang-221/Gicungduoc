## ⚡ Quick Commands

```bash
# Clone starter
git clone https://github.com/ChauGiang-221/Gicungduoc.git

# Backend setup
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

# Frontend setup
cd frontend
npm install

# Deploy
vercel --prod          # Frontend
railway up             # Backend

# Test API
curl http://localhost:8000/health
curl http://localhost:8000/api/v1/ai/providers
```
